The Advanced Java Programming Course is a one-week course delivered by Prof. Khalid Mughal with Sserunjogi Richard serving as the Teaching Assistant from 5th to 9th November 2018.

It will cover several advanced java programming topics based on the interest of the participants

The prerequisite for attending this course is an understanding of object-oriented programming and java basics.

Endeavor to carry your laptop with Java Development Kit installed.

  • Introduction to Java Professional Certification
  • Using an Integrated Development Environment (IDE)
  • Using the JDK Tools (javac, java, javadoc, jshell) and the Java API Documentation
  • Encapsulation/Inheritance/Aggregation/Composition
  • Method Overriding and Method Overloading
  • Annotations
  • Advanced Class Design
  • Interfaces
  • Polymorphism (ls-13-polymorphism-interfaces)
  • Functional interfaces
  • Lambda expressions
  • Built-in functional interfaces
  • Object Equality, Comparable, Comparator (PGJ8_OCP_Ch06_OBJ_COMPARE_slides)
  • Dynamic Data Structures: lists, sets, and maps (OCA_Chap_10_slides, PGJ8_OCP_Ch06_COLLECTIONS_AND_MAPS_slides)
  • Sorting and Searching(ls-09-sorting-searching-arrays)
  • Collections and Arrays Utility Classes API
  • Exception Propagation
  • Exception Handling
  • Try-with-resources statement
  • Assertions
